# PHP Backend Project (Research Participation)

## Overview
This repository contains backend development work completed using plain PHP as part of a research participation role.  
The focus of this project was on the ease of implementing backend functionality within an existing web application.

## Scope of Work
The work in this repository primarily includes:

- Backend logic creating MVC architecture using plain PHP
- Controllers and routing
- Database interactions
- Configuration and application structure with PHP from scratch

## Technologies Used
- PHP
- MySQL

## Project Structure Notes

The frontend UI and initial database structure were provided as part of the research project.  
This repository focuses on backend-related work only. Frontend assets and sensitive configuration files have been excluded.
